## Top view
剣道の上達を目指すためのアプリです。

なぜ有効打突にならないのかわからないという課題を解決します。

自分の試合動画や稽古動画を投稿することで、自分の技を上級者と比較して客観的に見られます。また、比較して改善点を記録して公開することでアドバイスが得られる仕様になっております。
